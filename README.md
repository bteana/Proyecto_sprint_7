# Aplicación web de análisis de anuncios de coches usados

## Descripción del proyecto

Este proyecto consiste en el desarrollo y despliegue de una aplicación web interactiva para el análisis exploratorio de anuncios de venta de coches usados en Estados Unidos.
La aplicación fue desarrollada utilizando **Python** y la librería **Streamlit**, y permite visualizar información clave del conjunto de datos de manera sencilla e intuitiva.

## Funcionalidad de la aplicación

La aplicación web permite al usuario:

* Visualizar un **histograma** para analizar la distribución de una variable numérica del conjunto de datos.
* Visualizar un **gráfico de dispersión** para explorar la relación entre dos variables.
* Interactuar con la aplicación mediante **botones o casillas de verificación**, que controlan la generación de los gráficos.
* Explorar los datos de forma dinámica sin necesidad de escribir código.

Las visualizaciones se construyen utilizando la librería **plotly-express**, lo que permite gráficos interactivos directamente en el navegador.

## Tecnologías utilizadas

* Python
* Pandas
* Streamlit
* Plotly Express

## Estructura del proyecto

* `app.py`: archivo principal de la aplicación web.
* `vehicles_us.csv`: conjunto de datos utilizado para el análisis.
* `notebooks/EDA.ipynb`: notebook con el análisis exploratorio inicial.
* `requirements.txt`: lista de dependencias necesarias para ejecutar la aplicación.

## Despliegue

La aplicación se encuentra desplegada en la plataforma **Render** y puede ejecutarse desde un navegador web siguiendo el enlace https://proyecto-sprint-7-mdar.onrender.com
